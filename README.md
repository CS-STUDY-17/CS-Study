# 💎 CS-Study 💎
## 🍮 진행 방식
- 교재 1주일에 1챕터
- 해당 챕터 유형 6문제(골드 2문제 이상) + 책 연습문제
- 브론즈는 카운트 X
- 매주 주말 저녁 > 리뷰 + 다음 주 문제 선별
- 코드리뷰 전까지 제출
- 벌금 문제당
    - 아예 못 풀었을 시 3000원
    - 풀다가 다 못 풀면 1000원

## 🍦 디렉토리 구조 및 파일명
```
├──{주제 이름}
│  ├──{깃허브 아이디}
│  │  ├──boj_{문제번호}.py
└── ──└──book_{챕터번호}_{문제번호}.py
```
